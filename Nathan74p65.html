2) a) Exprimer le cot de chaque contrat en fonction de x. Et reprsenter graphiquement les deux fonctions obtenues dans un mme repre.

On a $\;f_1(x)=0,4\times x+23\;$ et $\;f_2(x)=0,6\times x\;$

2) b) Graphiquement: Jusqu' 115 km le contrat $C_2$ est le plus avantageux.

Remarque : on peut aussi utiliser la librairie sympy (calcul symbolique) pour rsoudre l'quation $\;f_1(x)=f_2(x)\;$ avec la commande solve.

In [19]:
from sympy import *
x = Symbol('x')
#init_printing(use_latex=True)
solve(Eq(0.4*x+23, 0.6*x), x)
Out[19]:
$$\left [ 115.0\right ]$$

2) c) Ecrire un algorithme dont l'objectif est de donner, suivant le nombre x de kilomtres parcourir, le nom du contrat le moins cher et le prix payer.

In [20]:
x=float(input('Saisir le nbre de km : '))
if 0<=x and x<=115:
    P=0.6*x
    contrat='C1'
else:
    P=0.4*x+23
    contrat='C2'
print("Le contrat le moins cher est : ", contrat)
print("Le montant  payer sera de : ", P, "euros")
